Examples
########

The following examples demonstrate how to use Magrittetorch.

The corresponding jupyter notebooks can be found in the source code at :code:`docs/src/1_examples/`.
They can be run using jupyterlab, which is included in the Magrittetorch conda environment.
The documentation on how to post-process hydrodynamic models is still under construction.
